# MJC_in_one

명지전문대학 재학생을 위한 **통합 공지사항 제공 모바일 앱 개발**을 목표로 하는 프로젝트입니다. 
파편화된 학교 공지사항(메인, 교수학습, 역량관리)정보를 한곳에서 볼 수 있도록 만드는 것을 지향합니다.

## 프로젝트 개요

- **소속**: MJC AI-DEAS 교내 프로그램 지원 하에 진행
- **방향**: 3개 사이트 공지사항 통합, 도서관 홈페이지 웹뷰 구현, 키워드 알람 구현 등등

## 관련 구성 (상위 워크스페이스)

- Flutter 기반 클라이언트 앱 (`app/`)
- 학교 게시판 등 데이터 수집·동기화용 스크립트 (`test/crawler_mjc.py` 등) 및 Firebase 연동

개발 진척도는 PROGRESS.md 에서 확인 가능합니다.
