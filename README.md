# MJC_in_one

명지전문대학 재학생·통학생을 위한 **모바일 앱 개발**을 목표로 하는 프로젝트입니다. 학교 생활과 이동(통학)에 필요한 정보를 한곳에서 볼 수 있도록 만드는 것을 지향합니다.

## 프로젝트 개요

- **소속**: MJC AI-DEAS 등 교내 프로젝트 맥락에서 진행
- **방향**: 공지·학사 안내 등 캠퍼스 정보를 앱으로 제공하고, 통학·캠퍼스 이동과 연계된 기능을 단계적으로 확장하는 구조

## 관련 구성 (상위 워크스페이스)

- Flutter 기반 클라이언트 앱 (`app/`)
- 학교 게시판 등 데이터 수집·동기화용 스크립트 (`test/crawler_mjc.py` 등) 및 Firebase 연동

이 저장소(`MJC_in_one`)는 위 내용을 아우르는 **프로젝트 단위 저장소**로, 통학 앱 전체 기획·연동을 정리하는 용도로 사용할 수 있습니다.
